# NodeJs-SMS

> <Subtitle>
> Send SMS via NodeJs

---

## Features

- Easily send SMS via the Twilio API
- Easily send SMS via the Nexmo API

## Instructions to run

```
$ git clone https://github.com/RajatSablok/nodejs-sms.git
$ npm install
$ npm start

```
